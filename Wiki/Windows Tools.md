# 🪟 Windows Tools
### Useful tools for Windows. 

***

### Activate Windows and Microsoft Office:

##### 1. [KMS_VL_ALL_A10 ⭐ ](https://github.com/abbodi1406/KMS_VL_ALL_AIO/releases/tag/v0.49.0)
KMS_VL_ALL_A10 is a smart activation script that is easy to use and is smart enough to not overwrite already activated products.

##### 2. [MAS ⭐ ](https://github.com/massgravel/Microsoft-Activation-Scripts)
A Windows and Office activator using HWID / KMS38 / Online KMS activation methods, with a focus on open-source code and fewer antivirus detections.
